# 复盘报告质量自查

每份报告人工打 0/1：

- [ ] 大盘段包含具体点位数值（如 3200、创业板 2800）
- [ ] 大盘段包含涨跌幅 / 量能变化百分比
- [ ] 大盘段 ≤ 300 字
- [ ] 板块段每条都有：走势 + 逻辑 + 持续性 + 操作建议 + 避雷
- [ ] 板块段字数：普通板块 ≤ 200 字，重点板块（字幕多次提及）≤ 300 字
- [ ] 买卖纪律段有具体仓位比例或触发条件
- [ ] 保留主播原话感（不是 AI 改写味）
- [ ] 全文无 emoji / 无表格 / 无多余寒暄
- [ ] 外围联动 / 后市展望：字幕未提及时已省略（不硬凑）
- [ ] 无脑补内容（所有结论可在字幕中找到出处）

达标 ≥ 8/10 算合格。不达标 → 记录失败模式，周末统一调整 prompt 或 hotwords。
